# Rule Text Analysis:

The following are the observed patterns of rule text in the given rules dataset:

### Abstracts containing a keyword or a keyword family. [may contain excluding criteria]

- All abstracts with mentions of competitors or research keywords.
- Abstracts with tumor types without any genotypes

### Abstracts containing a keyword or a keyword family with a combination of other keyword or other keyword family [may contain excluding criteria]

- Abstracts with tumor types in combination with client specified genotypes excluding (K)RAS Wildtype

### A column containing a specific keyword.

- firms column containing Calliditas.

### Abstracts containing catalyst keywords.

- Abstracts that include any of the following: Phase 2/3 data (including meta-analyses), real-world data (including patient reported outcomes, retrospective analyses, and registry data).

### Abstracts containing catalyst keywords with combination of other keywords.

- Phase 2 data, Phase 3 data (including meta-analyses), real-world data (including PROs and registry data), Phosphate binders/phosphate lowering treatment.
- Phase 2 data, Phase 3 data (including meta-analyses), real-world data (including PROs and registry data), IBS-C, IBS (QOL, patient insights, new treatments), microbiota, and probiotics.

### Abstracts with presence of data in a specific column and absence of data in other columns.

- All abstracts that only have mention of an indication and do not include any other data.

### Abstracts with structural criteria

- Abstracts Containing More than 3 High Priority Keywords can be considered as Very High Priority.
